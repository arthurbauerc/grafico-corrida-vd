# Sertão Running 2023 – 8km Masculino

Aplicação Django que lê o arquivo de resultados da prova Sertão Running 2023 (8km masculino)  
e exibe um gráfico de linha interativo com os tempos dos atletas.


## Passo a passo para rodar o projeto

python -m venv .venv

.\.venv\Scripts\activate

cd .\sertao_running\

python manage.py migrate

python manage.py runserver
